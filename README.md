## Dependencies

## Installation
- Install Python 3.x
- Download Repo
- Create Virtual Environment. e.g. python3 -m venv ./venv
- Activate Environment. e.g source venv/bin/activate
- pip install -r requirements.txt
- Done



## Usage

1. python start.py
2. Choose your preferred Actions
3. Follow the Action steps
